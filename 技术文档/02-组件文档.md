# BookVoyage 组件文档

## 组件总览

```
App.tsx (根组件, 状态中心)
├── 认证面板 (内联, 登录/注册)
├── 导航标签栏 (Shelf | Global Map | Library Square | Reading Desk)
├── BookShelf3D.tsx      — 3D 书架视图
├── WorldMap.tsx         — D3 世界地图
├── CommunitySquare.tsx  — 社区广场
├── UserProfileDashboard.tsx — 个人中心
├── BookDetailModal.tsx  — 图书详情弹窗 (条件渲染)
├── BookReader.tsx       — 全屏阅读器 (条件渲染)
├── ImportBookForm.tsx   — 图书导入表单 (条件渲染)
└── ReviewForm.tsx       — 书评编辑表单 (条件渲染)
```

---

## 1. App.tsx — 根组件

**路径**: `src/App.tsx` | **行数**: ~803 | **角色**: 状态管理中心

### Props
无（React 根组件）

### 核心状态
| 状态 | 类型 | 说明 |
|------|------|------|
| `activeTab` | `'shelf'\|'map'\|'public'\|'profile'` | 当前激活的标签页 |
| `books` | `Book[]` | 全部图书列表 (从 API 加载) |
| `reviews` | `Review[]` | 全部书评列表 |
| `bookmarks` | `Bookmark[]` | 全部书签列表 |
| `currentUser` | `User\|null` | 当前登录用户 |
| `isLoggedOut` | `boolean` | 是否处于登出状态 |
| `selectedBook` | `Book\|null` | 当前选中的图书 (打开详情弹窗) |
| `readingBook` | `Book\|null` | 当前正在阅读的图书 |
| `isImportOpen` | `boolean` | 导入表单是否打开 |
| `isReviewOpen` | `boolean` | 书评表单是否打开 |

### 核心方法
| 方法 | 说明 |
|------|------|
| `handleImportBook(data)` | 导入新图书 → 调用 API → 更新 books 状态 |
| `handleDeleteBook(id)` | 删除图书 → 级联删除关联书评+书签 |
| `handlePublishReview(data)` | 发布书评 → 调用 API |
| `handleCloseReader(seconds, progress, page)` | 关闭阅读器 → 保存阅读进度 |
| `handleAddBookmark(position, chapter, note, snippet)` | 添加书签 |
| `handleDeleteBookmark(id)` | 删除书签 |
| `handleRegister(e)` | 注册新用户 → 调用 POST /api/auth/register |
| `handleLogin(email, password)` | 登录 → 调用 POST /api/auth/login |
| `handleGuestLogin()` | 游客登录 (绕过认证) |

### 生命周期
1. **挂载**: 检查 localStorage 中的 JWT token → 调用 GET /api/auth/me 验证 → 加载 books/reviews/bookmarks
2. **状态变更**: 每次修改 books/reviews/bookmarks 后自动同步到 API
3. **认证过期**: API 返回 401 → 清除 token → 显示登录界面

## 2. WorldMap.tsx — D3 交互式世界地图

**路径**: `src/components/WorldMap.tsx` | **行数**: ~651 | **依赖**: d3, topojson-client

### Props
```typescript
{
  books: Book[];
  reviews: Review[];
  activeBookId: string | null;  // 地图聚焦的图书ID
  onBookClick: (book: Book) => void;
}
```

### 内部状态
| 状态 | 说明 |
|------|------|
| `mapMode` | `'author_origin'` (作者国籍) \| `'reader_anchor'` (读者足迹) |
| `zoom` | D3 zoom 变换状态 |
| `selectedCountryCode` | 当前选中的国家 ISO 代码 |
| `hoveredCountryCode` | 当前悬停的国家 |
| `geoJsonData` | 从 CDN 加载的世界地图 GeoJSON |

### 核心功能
- **GeoJSON 渲染**: 加载 110m 精度世界地图，D3 GeoMercator 投影
- **双模式切换**: 作者起源模式 / 读者足迹模式
- **国家着色**: 23 种颜色映射，有数据的国家高亮
- **交互**: 缩放(按钮+滚轮)、拖拽平移、国家悬停 tooltip、点击侧边栏
- **标记点**: 脉冲发光圆点(国家中心) + 地图钉 SVG 图标(书评位置)
- **侧边面板**: 点击国家弹出图书列表/书评列表

### 外部依赖
- GeoJSON 数据: `https://cdn.jsdelivr.net/npm/world-atlas@2/countries-110m.json`
- TopoJSON 转换: `topojson-client` feature() 方法

## 3. BookShelf3D.tsx — 3D 书架

**路径**: `src/components/BookShelf3D.tsx` | **行数**: ~341

### Props
```typescript
{
  books: Book[];
  onBookClick: (book: Book) => void;
  onMoveBook: (bookId: string, newCategory: string) => void;
  onDeleteBook: (bookId: string) => void;
}
```

### 核心功能
- **分类分组**: 按 category 将图书分组到不同书架
- **3D 视觉效果**: CSS 3D transforms (perspective, rotateY, translateZ)
- **拖拽排序**: HTML5 Drag & Drop API，支持跨分类移动
- **删除**: 拖拽到浮动垃圾桶图标触发删除确认
- **进度标识**: 封面上的百分比丝带标记
- **封面生成**: 优先使用 Unsplash 图片，回退到程序化 SVG 封面
- **空状态**: 虚线占位符提示空书架

## 4. BookReader.tsx — 全屏阅读器

**路径**: `src/components/BookReader.tsx` | **行数**: ~562

### Props
```typescript
{
  book: Book;
  bookmarks: Bookmark[];
  onBack: (sessionSeconds: number, progress: number, page: number) => void;
  onAddBookmark: (position: number, chapterTitle: string, note: string, snippet: string) => void;
  onDeleteBookmark: (id: string) => void;
}
```

### 内部状态
| 状态 | 说明 |
|------|------|
| `currentChapterIndex` | 当前章节索引 |
| `currentPage` | 当前页码 (每章内) |
| `fontSize` | 字体大小 (12-24px) |
| `theme` | 主题: `parchment` \| `dark` \| `sepia` \| `light` |
| `readingStartTime` | 阅读开始时间戳 |
| `showBookmarkDrawer` | 书签抽屉是否打开 |
| `showSettings` | 设置面板是否打开 |

### 核心功能
- **章节分页**: 每 ~800 字符一页，保持换行完整性
- **双页展开**: 桌面端 (>md) 显示左右双页 + 书脊模拟
- **4 种主题**: 羊皮纸 / 深色 / 复古 / 浅色
- **字体调节**: 12-24px 范围滑块
- **书签系统**: 侧边抽屉添加/查看/删除/跳转书签
- **阅读计时**: Date.now() 差值 + visibilitychange 事件处理
- **翻页动画**: Framer Motion 过渡效果

## 5. ImportBookForm.tsx — 图书导入

**路径**: `src/components/ImportBookForm.tsx` | **行数**: ~533

### Props
```typescript
{
  onClose: () => void;
  onImport: (bookData: {...}) => void;
}
```

### 内部状态
| 状态 | 说明 |
|------|------|
| `file` | 当前选中的文件 |
| `content` | 文件文本内容 |
| `title/author/country/category/summary` | 表单字段 |
| `coverType` | `'procedural'` \| `'uploaded'` \| `'url'` |
| `selectedCoverSvg` | 程序化封面的 SVG data URI |
| `isExtracting` | AI 元数据提取加载中 |

### 核心功能
- **文件上传**: 拖拽 + 点击，支持 .txt/.epub/.pdf
- **文件名解析**: 从文件名提取作者和书名
- **AI 元数据提取**: 调用 POST /api/gemini/extract 自动填充表单
- **5 种程序化封面**: Classic Gold, Midnight Star, Botanical Ivy, Modern Slate, Desert Clay
- **自定义封面**: URL 输入或本地图片上传 (FileReader → base64)
- **表单校验**: 书名和作者为必填项

## 6. ReviewForm.tsx — 书评编写

**路径**: `src/components/ReviewForm.tsx` | **行数**: ~286

### Props
```typescript
{
  book: Book;
  onClose: () => void;
  onSubmit: (reviewData: {...}) => void;
}
```

### 内部状态
| 状态 | 说明 |
|------|------|
| `score` | 评分 1-5 |
| `title` | 书评标题 |
| `content` | 书评内容 |
| `selectedCountryName` | 选中的国家名 |
| `cityInput` | 城市输入 |
| `visibility` | `'private'` \| `'public'` |
| `isAiGenerating` | AI 生成加载中 |

### 核心功能
- **五星评分**: 可点击的星级选择器
- **AI 辅助写作**: "✨" 按钮调用 POST /api/gemini/review 生成书评
- **地理位置**: 国家下拉选择 (23 国) + 城市文本输入 + 坐标自动查找
- **坐标抖动**: 同国家多条书评坐标加随机偏移 (±0.4°)，避免重叠
- **可见性切换**: 公开 (🌐) / 私有 (🔒) 按钮
